# Go Rename Files CLI

This is a tool to rename files from the 'PascalCase' or 'camelCase to 'kebab-case'.
It is a CLI tool written in Go, using Cobra and promptui,

## Usage

1. Clone the repo
2. Install the dependencies
3. Run the CLI

```go
// at the root of the project
go run main.go
```
